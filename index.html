01 // IGORCADEMIA

HTML & CSS

Frontend desde cero con ejercicios interactivos, editores en vivo, retos visuales, IgorPoints, logros y una ruta pensada para construir páginas reales. Más de 70 ejercicios organizados en 17 módulos con sistema de premios y responsive desde el primer día.

70+ ejercicios17 módulos5 premiosResponsive
  1. Estructura HTML: DOCTYPE, html, head, body, jerarquía, metadatos y organización base. Aprenderás a escribir documentos HTML válidos desde cero.
  2. Semántica: header, nav, main, article, section, footer y buenas prácticas de accesibilidad. El código que entienden tanto los navegadores como Google.
  3. Formularios: form, input, select, textarea, atributos, validación nativa y formularios completos y funcionales.
  4. CSS base: selectores, clases, IDs, combinadores, box model, colores HEX/RGB/HSL y gradientes. La lógica que hay detrás de los estilos.
  5. Layout: display, position, Flexbox, CSS Grid, áreas, auto-fit y diseños adaptables a cualquier pantalla.
  6. Interacción visual: pseudoclases, hover, focus, transiciones, transformaciones y animaciones con keyframes.
  7. Responsive: media queries, mobile-first, unidades fluidas (clamp, vw, vh), variables CSS y temas de color dinámicos.
  8. Extras visuales: filter, backdrop-filter, Google Fonts, iconos, clip-path, mix-blend-mode y efectos visuales avanzados para páginas de nivel profesional.
02 // IGORSCRIPT

JavaScript

Programación web práctica: lógica, DOM, eventos, asincronía, APIs y proyectos visuales con datos reales. 124 ejercicios graduales que van de variables básicas hasta conectar con APIs externas como PokéAPI y SuperHero API.

124 ejercicios12 módulosAPIs realesProyectos
  1. Variables y tipos: var, let, const, strings, numbers, booleanos, null, undefined y template literals. La base de cualquier programa.
  2. DOM y selectores: getElementById, querySelector, innerHTML, textContent y atributos data-*. Aprende a manipular la página con código.
  3. Eventos: click, input, submit, teclado, addEventListener y comportamiento interactivo. Haz que la web responda al usuario.
  4. Lógica: condicionales, switch, ternarios, operadores lógicos, bucles for/while y control de flujo completo.
  5. Funciones: funciones clásicas, arrow functions, parámetros, return, scope y reutilización de código.
  6. Datos complejos: arrays, objetos, push, map, filter, find, spread, destructuring y estructuras de datos modernas.
  7. Estilos con JS: classList, style, toggles, render dinámico y animaciones disparadas por código sin tocar CSS.
  8. Async y proyectos: fetch, async/await, PokéAPI, SuperHero API, cards dinámicas y buscadores visuales conectados a datos reales.
03 // IGORHARDWARE

Hardware

Sistemas microinformáticos con estética tierra/ocre: componentes, montaje, diagnóstico, compatibilidad y laboratorios prácticos. Aprende a construir un PC desde cero, resolver averías reales y entender cada pieza que hay dentro de una máquina.

PC BuilderIncidenciasLaboratorio 2026Flashcards
  1. Fundamentos: unidades de medida, hardware vs software, tipos de ordenadores y periféricos. La base conceptual antes de tocar ninguna pieza.
  2. Componentes: CPU, placa base, RAM, almacenamiento (SSD/HDD/NVMe), GPU, fuente de alimentación, refrigeración y cajas.
  3. Firmware y sistema: BIOS/UEFI, POST, sistema operativo, Windows, instalación de drivers y conceptos de virtualización.
  4. Montaje: orden de ensamblaje, cableado modular, ESD, medidas de seguridad, prueba de encendido y errores comunes al montar.
  5. Compatibilidad: sockets de CPU, chipset, formato de RAM, consumo TDP, cuello de botella y equilibrio de presupuesto real.
  6. Diagnóstico: equipos que no arrancan, pantallas negras, pitidos BIOS, ruidos, temperaturas elevadas y fallos de red.
  7. Simuladores: PC Builder con catálogo de piezas real para practicar presupuestos y un resolutorio de incidencias paso a paso.
  8. Laboratorio moderno: IA integrada en PCs (NPU, Copilot+), tendencias 2026, nuevas arquitecturas y tecnologías emergentes.
04 // IGORREDES

Redes

Networking gamificado para entender conectividad, direccionamiento, subnetting, WiFi, VLANs, routing y topologías. Aprende redes con simuladores visuales, calculadoras interactivas y un configurador de router estilo Cisco IOS.

Simulador de redSubnettingRouter CLIMapas lógicos
  1. Fundamentos: conceptos de red, modelos OSI y TCP/IP, tipos de cableado, capa física y diseño de red básico.
  2. Direccionamiento IP: IPv4, máscaras de subred, clases de red, hosts, gateway, DNS y configuración práctica de red.
  3. Subnetting: calculadora visual, divisor de subredes, ejercicios resueltos y generador de subnetting aleatorio para practicar.
  4. Red inalámbrica: WiFi 5/6/6E, cobertura, seguridad WPA3, tendencias actuales y laboratorio inalámbrico 2026.
  5. VLAN y avanzado: segmentación lógica, routing entre VLANs, switching y escenarios de red empresarial real.
  6. Simulador de red: arrastra dispositivos, conéctalos, configura IPs, guarda topologías y haz ping entre máquinas virtuales.
  7. Mapas lógicos: editor visual de planta para trazar paredes, zonas, cableado y dispositivos de red en un espacio real.
  8. Router CLI: configurador con sintaxis Cisco IOS, lecciones guiadas paso a paso y modo libre para experimentar.
05 // IGORSERVER

Server

Administración de sistemas con Windows Server, Linux, servicios de red, virtualización, copias de seguridad y monitorización. Trabaja con escenarios reales de empresa: dominios, usuarios, roles y políticas de grupo.

Windows ServerActive DirectoryPowerShellNagios
  1. Windows Server: introducción, roles de servidor, administración remota, escritorio remoto y configuración de escenarios reales.
  2. Active Directory: dominio, usuarios, grupos, unidades organizativas (OU), permisos y estructura de empresa completa.
  3. DNS y DHCP: resolución de nombres, zonas, registros, concesiones DHCP, reservas y servicios esenciales de red.
  4. VirtualBox: creación de laboratorios locales, máquinas virtuales, redes internas, snapshots y simuladores para practicar sin riesgo.
  5. Almacenamiento: discos, particiones, volúmenes dinámicos, RAID 0/1/5/10 y administración segura de datos.
  6. Backups: estrategias de copia de seguridad, Acronis True Image, recovery, pruebas de restauración y simulador de recuperación.
  7. Avanzado: GPOs, políticas de contraseñas, PowerShell Server, auditoría de seguridad, permisos NTFS y redes avanzadas.
  8. Linux y monitorización: Ubuntu Server, integración Samba + AD, distintas distribuciones Linux y monitorización con Nagios.
06 // NUEVA RUTA

Ciberseguridad

Una ruta preparada para aprender seguridad digital con mentalidad defensiva, ética y práctica: proteger sistemas, entender riesgos, responder ante incidentes y aplicar medidas de seguridad en entornos reales. Ruta en construcción.

DefensaRiesgosLaboratoriosÉtica
  1. Fundamentos: amenazas, vulnerabilidades, impacto, la tríada CIA (confidencialidad, integridad, disponibilidad) y tipos de ataques.
  2. Higiene digital: contraseñas fuertes, gestores de contraseñas, MFA, copias de seguridad, privacidad online y actualizaciones.
  3. Redes seguras: firewall, segmentación de red, puertos abiertos, servicios expuestos y detección básica de intrusiones.
  4. Hardening: reducción de superficie de ataque, permisos mínimos, usuarios y configuración segura de sistemas.
  5. Ingeniería social: phishing, smishing, vishing, suplantación de identidad, señales de alerta y protocolos de respuesta.
  6. Análisis de riesgos: inventario de activos, criticidad, probabilidad, impacto y priorización de medidas de seguridad.
  7. Respuesta a incidentes: recopilación de logs, aislamiento, documentación, recuperación de sistemas y lecciones aprendidas.
  8. Laboratorios éticos: escenarios guiados, checklist de seguridad, generación de reportes y práctica siempre dentro del marco legal.
07 // IGOR SQL ACADEMY

SQL y Bases de Datos

Bases de datos relacionales con SQLite, diagramas entidad-relación, consola SQL real integrada en la web, bases de datos temáticas y ejercicios progresivos que van de lo más básico hasta subconsultas y transacciones.

SQLiteConsola realBDs temáticasDiagramas ER
  1. Fundamentos: tablas, registros, columnas, relaciones, SGBD, modelo relacional y por qué usar una base de datos.
  2. Herramientas: DB Browser, DBeaver, SQLiteOnline y DataGrip. Aprende el flujo de trabajo de un administrador de BBDD.
  3. Modelo de datos: tipos de datos, restricciones NOT NULL/UNIQUE, primary key, foreign key, cascadas y cardinalidad.
  4. DDL y DML: CREATE TABLE, INSERT, UPDATE, DELETE, DROP y ALTER TABLE. Crea y modifica estructuras de datos reales.
  5. Consultas: SELECT, WHERE, alias, DISTINCT, ORDER BY, LIMIT, LIKE, BETWEEN, IN y combinaciones complejas.
  6. Agregaciones: COUNT, SUM, AVG, MIN, MAX, GROUP BY y HAVING para resumir y analizar datos agrupados.
  7. Avanzado: INNER/LEFT/RIGHT JOINs, subconsultas, normalización 1FN-3FN, vistas, transacciones e índices de rendimiento.
  8. Práctica temática: bases de datos de Pokémon, Marvel, hospital, streaming, colegio, baloncesto y laboratorio SQL libre.

Excel

Domina la herramienta ofimática más usada en el mundo empresarial: fórmulas, funciones, tablas dinámicas, gráficos, Power Query, automatización con macros VBA y análisis de datos profesional. Desde cero hasta nivel avanzado con ejercicios prácticos basados en casos reales.

FórmulasTablas dinámicasPower QueryPróximamente
  1. Fundamentos: interfaz, celdas, rangos, tipos de datos, formatos, atajos de teclado y navegación eficiente. La base para trabajar rápido.
  2. Fórmulas esenciales: operadores, referencias relativas y absolutas ($), SUMA, PROMEDIO, CONTAR, MAX, MIN, SI, Y, O y anidamiento de funciones.
  3. Funciones avanzadas: BUSCARV, BUSCARX, INDICE+COINCIDIR, SUMAR.SI.CONJUNTO, CONTAR.SI, TEXTO, FECHA y funciones de búsqueda y referencia.
  4. Tablas y listas: convertir rangos a tablas inteligentes, filtros, ordenación, segmentación y referencias estructuradas para fórmulas dinámicas.
  5. Tablas dinámicas: crear, configurar, agrupar, filtrar con segmentadores, calcular campos y mostrar datos desde múltiples ángulos sin una sola fórmula.
  6. Gráficos y dashboards: tipos de gráfico, formato profesional, gráficos combinados, minigráficos (sparklines) y paneles de control interactivos.
  7. Power Query: importar datos de Excel, CSV, web y bases de datos; limpiar, transformar, combinar y actualizar datos automáticamente sin código.
  8. Macros y VBA: grabadora de macros, editor VBA, bucles, condicionales, formularios de usuario y automatización de tareas repetitivas con un clic.

PromptsAsistentesAutomatizaciónProyectos

10 // PRÓXIMAMENTE

Linux

Domina el sistema operativo que mueve Internet: terminal, permisos, scripting con Bash, administración de servicios y las distribuciones más usadas en producción. Desde los primeros comandos hasta automatizaciones completas con shell scripting.

TerminalBash scriptingDistrosPróximamente
  1. Introducción: qué es Linux, filosofía Unix, distribuciones (Ubuntu, Debian, Fedora, Arch), escritorios y cuándo elegir cada una.
  2. Terminal básica: ls, cd, pwd, mkdir, rm, cp, mv, cat, less, nano. Navegar y gestionar el sistema desde la línea de comandos.
  3. Permisos: usuarios, grupos, chmod, chown, umask, sudo y la lógica de seguridad de Linux.
  4. Gestión de paquetes: apt, dnf, pacman, snap, flatpak. Instalar, actualizar y eliminar software desde terminal.
  5. Procesos y servicios: ps, top, htop, kill, systemctl, journalctl y gestión de servicios con systemd.
  6. Red en Linux: ip, ping, netstat, ss, ssh, scp, rsync, UFW y configuración básica de interfaces de red.
  7. Bash scripting: variables, condicionales, bucles, funciones, argumentos, cron y automatización de tareas repetitivas.
  8. Administración avanzada: LVM, RAID software, logs del sistema, monitorización con htop/Glances y hardening básico de Linux.

Python

El lenguaje más versátil del momento: desde los primeros scripts hasta proyectos de automatización, análisis de datos, APIs REST y backend. Aprende Python con proyectos reales aplicados a problemas que importan.

ScriptingDatosAPIsPróximamente

ComponentesHooksEstadoPróximamente
